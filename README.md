# Tetris-game
A classic tetris game implemented using HTML5 canvas and JavaScript.

Game features: 
* Multiple difficulty levels,
* Score counting,
* Music/Sounds,
* Timer,
* Game pause/resume.

Play https://ewelina90.github.io/Tetris-game/

## For this project I have used:
* JavaScript
* HTML5 Canvas
* SASS
* Webpack
